# kzcf

E站APP - 汽车配件商城

## 项目说明

这是一个交互式的汽车配件商城应用预览页面。

## 主要功能

### 1. 首页
- 搜索栏和位置信息
- 快捷功能菜单
- 门店管理功能
- 商品分类展示

### 2. AI报价（IM聊天界面）
- 智能对话机器人
- 输入车架号即可获取配件报价
- 实时消息显示
- 自动回复配件信息和价格

### 3. 个人中心
- 用户信息管理
- 订单管理
- 门店管理

## 部署

项目已部署到Vercel，访问 https://kzcf.vercel.app 即可查看。

## 文件说明

- `index.html` - 应用主页面，包含完整的UI和交互功能

## 使用说明

1. 点击底部导航栏的"AI报价"按钮进入聊天界面
2. 在输入框中输入17位车架号（例如：LE4TG4DB9KL243278）
3. AI助手会自动返回对应车辆的配件报价信息
