# kzcf

E站APP - 汽车配件商城

## 项目说明

这是一个交互式的汽车配件商城应用预览页面，包含完整的登录流程和主应用功能。

## 文件说明

### 登录流程页面
- `login.html` - 初始登录页面，支持本机号码一键登录
- `phone-login.html` - 手机号输入页面，手动输入手机号
- `verify-code.html` - 验证码输入页面，带数字键盘

### 主应用页面
- `index.html` - 应用主页面，包含完整的UI和交互功能

## 主要功能

### 登录功能
1. **本机号码一键登录**
   - 显示本机号码（脱敏显示）
   - 隐私协议勾选
   - 一键登录按钮

2. **手机验证登录**
   - 手机号输入（+86前缀）
   - 实时输入验证
   - 自动激活登录按钮

3. **验证码验证**
   - 4位验证码输入
   - 自定义数字键盘
   - 自动验证功能
   - 倒计时提示

### 应用主要功能

#### 1. 首页
- 搜索栏和位置信息
- 快捷功能菜单
- 门店管理功能
- 商品分类展示

#### 2. AI报价（IM聊天界面）
- 智能对话机器人
- 输入车架号即可获取配件报价
- 实时消息显示
- 自动回复配件信息和价格

#### 3. 个人中心
- 用户信息管理
- 订单管理
- 门店管理

#### 4. 设置页面
- 收货地址编辑
- 手机号修改
- 收货人修改
- 地图选点功能

## 使用说明

### 登录流程
1. 访问 `login.html` 进入登录页面
2. 勾选隐私协议后点击"本机号码一键登录"
3. 或点击"切换手机号"手动输入手机号
4. 在验证码页面输入4位验证码
5. 自动跳转到应用主页

### 应用使用
1. 点击底部导航栏的"AI报价"按钮进入聊天界面
2. 在输入框中输入17位车架号（例如：LE4TG4DB9KL243278）
3. AI助手会自动返回对应车辆的配件报价信息
4. 在个人中心可以管理订单和门店信息
